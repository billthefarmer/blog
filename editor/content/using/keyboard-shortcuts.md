When using an external keyboard, some keyboard shortcuts are
implemented:
 * Ctrl+E &ndash; Edit mode
 * Ctrl+N &ndash; New file
 * Ctrl+O &ndash; Open file
 * Ctrl+S &ndash; Save file
 * Ctrl+Shift+S &ndash; Edit
 * Ctrl+V &ndash; View mode
